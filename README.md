# Deepak-Waman.github.io

Personal academic website of Dr. Deepak Waman, postdoctoral researcher in cloud microphysics at KIT.

Built with [Jekyll](https://jekyllrb.com) and the [Minimal Mistakes](https://mademistakes.com/work/jekyll-themes/minimal-mistakes/) theme.

## Site Structure

- `_config.yml` -- site settings, author info, navigation
- `index.md` -- home page
- `_pages/research.md` -- research overview
- `_pages/publications.md` -- publication list
- `_pages/projects.md` -- active projects
- `_pages/contact.md` -- contact information
- `assets/images/profile.jpg` -- your profile photo (add this manually)
- `assets/files/CV_Deepak_Waman.pdf` -- your CV (add this manually)

## To Update

Edit the relevant `.md` file and commit. GitHub Pages rebuilds the site automatically within a minute or two.
